Hey visitor! It seems that you like to use the keyboard to scroll. Please consider upgrading to (probably) better shortcuts.

  • >: go bottom.

  • j: scroll next.
  • k: scroll previous.
  • l: forward section.
  • h: backward section.
  • g: go top.
  • G: go bottom.
  • Others
  • d: scroll down.
  • u: scroll up.
  • m: toggle navigation.
  • space: toggle animation.
  • arrows: prev/next background.
